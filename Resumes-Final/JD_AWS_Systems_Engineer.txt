Job Title: AWS Systems Engineer (Mid-Level)
Location: Bengaluru, India (Hybrid)
Experience: 2–4 Years

About the Role:
We are hiring a mid-level AWS Systems Engineer to own and operate our cloud and hybrid infrastructure.
You will be responsible for designing, deploying, and maintaining secure, scalable AWS environments
and ensuring high availability across our production systems.

Key Responsibilities:
- Design and manage AWS infrastructure: EC2, S3, RDS, Lambda, VPC, IAM, CloudWatch, CloudFormation
- Administer Linux (Ubuntu/CentOS/RHEL) and Windows servers in production
- Implement network security: firewalls, VPN, security groups, NACLs, load balancers
- Write automation scripts in Bash and/or Python for routine ops tasks
- Manage backup strategies and disaster recovery across regions
- Enforce IAM policies, security compliance, and access controls
- Support CI/CD pipelines (Jenkins, GitHub Actions, or similar)
- Provide L3 support and conduct root cause analysis for production incidents
- Maintain infrastructure documentation and architecture diagrams

Mandatory Requirements:
- 2+ years hands-on AWS production experience (not just certifications)
- Proven Linux system administration in production environments
- Strong networking knowledge: TCP/IP, DNS, VPN, Load Balancers, Firewalls
- Scripting proficiency: Bash, Python, or PowerShell
- Experience with backup and DR planning

Preferred / Bonus:
- AWS Certification (Solutions Architect / SysOps / DevOps)
- Infrastructure as Code (Terraform or CloudFormation)
- Docker and/or Kubernetes experience
- CI/CD pipeline management
- Windows Server administration

Skills: aws, ec2, s3, rds, lambda, vpc, iam, cloudwatch, linux, networking, bash, python, security, devops
